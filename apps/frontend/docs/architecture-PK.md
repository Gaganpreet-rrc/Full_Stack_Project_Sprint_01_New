# Sprint 3 Architecture – PK

## Repository: libraryTipsRepository
Stores Library Tips data and provides CRUD methods using test data.

## Service: libraryTipsService
Handles business rules like validation and ID generation.

## Hook: useLibraryTips
Manages UI state and connects the component to the service.

## Component: LibraryTips
Uses the hook to display, add, and remove tips following the Sprint 3 architecture.
